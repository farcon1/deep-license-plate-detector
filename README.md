# Deep License Plate Detector

Магистерская ВКР по разработке системы детекции и локализации автомобильных номеров на изображениях с использованием глубоких нейронных сетей.

**Полное название ВКР (RU)**  
«Разработка системы детекции и локализации автомобильных номеров на изображениях с применением глубоких нейронных сетей»

**Полное название ВКР (EN)**  
«Development of a Deep Learning-Based System for Vehicle License Plate Detection and Localization»

---

## Цель проекта

Разработать и экспериментально обосновать систему детекции и локализации автомобильных номерных знаков, устойчивую к сложным условиям съёмки (ночь, движение, плохая погода, различные форматы номеров) и пригодную для практического применения.

## Ключевые задачи

- Анализ существующих подходов к автоматическому распознаванию автомобильных номеров (классические методы и нейросетевые детекторы).
- Подбор и подготовка открытых датасетов (очистка, унификация аннотаций, разбиение на train/val/test, выделение сложных случаев).
- Реализация базовых решений: классический компьютерно-зрительный pipeline и нейросетевая модель (YOLO/Faster R-CNN и др.) для детекции номерных знаков.
- Экспериментальное сравнение архитектур по точности, скорости и робастности к сложным условиям.
- Обучение и выбор финальной модели детекции и локализации автомобильных номеров.
- Разработка демонстрационного прототипа (например, Telegram-бота), выполняющего детекцию номеров на пользовательских изображениях.

## Технологический стек

- Python, PyTorch
- Современные детекторы объектов (семейства YOLO, Faster R-CNN, при необходимости SSD)
- Инструменты для аугментации изображений и анализа качества моделей
- Telegram Bot API или веб-сервис для демонстрации работы системы
